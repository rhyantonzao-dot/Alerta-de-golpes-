# Alerta-de-golpes-
Projeto em HTML5 para alertar sobre golpes digitais.
⚠️ ALERTA PÚBLICO ⚠️
Este site foi criado com o objetivo de alertar pessoas sobre plataformas digitais que utilizam estratégias de manipulação psicológica para aplicar golpes financeiros.

Nenhuma plataforma séria exige pagamento, depósito ou “nível VIP” para liberar dinheiro que supostamente já é seu.
Como o golpe funciona
✔️ Liberação de pequenos saques iniciais
✔️ Criação de confiança
✔️ Exibição de saldo alto fictício
✔️ Limite diário de saque
❌ Bloqueio do sistema
❌ Exigência de depósito ou nível VIP
Saldo alto não significa dinheiro real
Valores elevados exibidos na tela são, muitas vezes, apenas números criados para induzir a vítima a continuar depositando dinheiro.

Se o saque só é liberado após novo depósito, o dinheiro exibido não existe fora da tela.
Impacto real nas pessoas
Esse tipo de golpe causa prejuízos financeiros e danos emocionais profundos. Muitas vítimas entram em desespero, ansiedade e depressão.

Informação e prevenção salvam vidas.
